How to Save Your Open Graph Image

  1. Choose a template or start from scratch:
    • Minimal Template: Simple design with text and decorative elements
    • Devices Template: Complex design with laptop and mobile device mockups (requires external editing)
  2. Customize your design using the tools provided
  3. Click "Export as SVG" to download the SVG file
  4. Save the file in the "assets/images" folder with one of these names:
    • og-image-devices.png: For the minimal template
    • og-image-devices.svg: For the devices template
  5. Update the Open Graph meta tag in index.html to point to your chosen file if needed

Note: All pages now point to assets/images/og-image-devices.png
