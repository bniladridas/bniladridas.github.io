Face Recognition

Real-time face recognition project on youcam The idea is to detect and recognize the faces, live from a video stream in real-time, by using face database on mat file. In this particular example, the system is trained to detect Elon Musk. This includes video capture and the image processing from OpenCV, Video streaming in web interfacing, face detection/recognizing with the package :face_recognition.
